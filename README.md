 <p> 
 Mapty is a vanilla JavaScript application that use the Leaflet library and display Map that users can interact with and save their workouts on it in local storage. 
</p>

### Features

#### Displaying Map using Leaflet library

#### Position automatically obtained by the browser using Geolocation API

#### Create or remove workouts

#### saving user workouts in local storage
